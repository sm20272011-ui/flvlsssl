# 🎮 Jeu de Football 3D

Un jeu de football 3D interactif développé avec **Three.js** et **JavaScript**.

## 🚀 Fonctionnalités

- ✅ Terrain 3D réaliste avec herbe et lignes
- ✅ 10 joueurs (5 par équipe)
- ✅ Ballon physique avec gravité et rebond
- ✅ Contrôle du joueur à la souris et clavier
- ✅ Système de pointage automatique
- ✅ IA basique pour les joueurs non contrôlés
- ✅ Caméra dynamique suivant le joueur
- ✅ Effets de lumière et ombres

## 🎮 Contrôles

| Touche | Action |
|--------|--------|
| **WASD** ou **Flèches** | Déplacer le joueur |
| **Espace** ou **Clic souris** | Tirer/Passer le ballon |
| **R** | Réinitialiser le jeu |
| **Souris** | Regarder autour |

## 📦 Installation

1. Cloner le dépôt
2. Ouvrir `index.html` dans un navigateur moderne
3. Profiter du jeu ! ⚽

## 🛠️ Technologies

- **Three.js** : Moteur de rendu 3D
- **JavaScript** : Logique du jeu
- **WebGL** : Rendu graphique
